使用说明

这是纯 Electron 单机版，不依赖 Lovable，不打开网页。

上传到 GitHub 后，进入 Actions，运行 Build Windows EXE，完成后在 Artifacts 下载。

如果要重新上传源码，请确保根目录直接包含 package.json、main.js、index.html、.github 文件夹。
